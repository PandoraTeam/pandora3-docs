## Структура каталогов

### Core

Содержит следующие:

```
  ├── Application
  ├── Container
  ├── Controller
  ├── Debug
  ├── Http
  ├── Interfaces
  ├── Properties
  ├── Registry
  ├── Router
  ├── Widget
```

Класс `Application` содержит следующие файлы `Application.php`, `BaseApplication.php` , `ApplicationException.php`


### Libs

```
  ├── Console
  ├── Database
  ├── DomainModel
  ├── File
  ├── RewriteRouter
  ├── Session
  ├── Time
```
